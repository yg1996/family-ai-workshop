# ChatGPT — מעבר לשאלות (דף עזר לקחת הביתה)

<div dir="rtl">

## 3 השדרוגים (+ בונוס)

1. **הוא יכול להכיר אתכם** — מגדירים Custom Instructions פעם אחת; כל שיחה מתחילה אישית.
2. **הוא יכול לקרוא את הדברים שלכם** — מעלים PDF, תמונות, צילומי מסך; שואלים שאלות עליהם.
3. **הוא יכול לעבוד בשבילכם** — Projects שומרים את כל המוח של נושא במקום אחד; Tasks רצים לפי לוח זמנים בלעדיכם.
4. **בונוס: לשאול מומחים** — ב-GPT Store יש יותר מ-159,000 עוזרים מוכנים. התקנה בקליק.

## 6 השורות שכל אחד צריך להגדיר

איפה: **Settings ← Personalization ← Custom Instructions**. מעתיקים, ממלאים סוגריים, גומרים — כל שיחה משתפרת לתמיד:

</div>

```text
עליי: אני גר/ה ב[עיר], ישראל. אני לא טכני/ת. הטלפון שלי [דגם], המחשב שלי
[Windows/Mac]. [קבועים נוספים: למשל אנחנו שומרים כשרות].

איך לענות: תמיד ענה בעברית. קצר ופרקטי. בלי ז'רגון טכני — אם חייבים מונח
טכני, הסבר אותו במשפט אחד. כשאתה נותן הוראות: שלבים ממוספרים, פעולה אחת
בכל שלב, ותגיד לי בדיוק איפה ללחוץ. בהחלטות: 2-3 אפשרויות עם יתרונות
וחסרונות, ואז המלצה. מחירים בשקלים. אם השאלה שלי לא ברורה — שאל אותי
לפני שאתה עונה.
```

<div dir="rtl">

**כלל המיון:** מה שתמיד נכון עליכם ← להוראות (Instructions). מה ששייך לשאלה של היום ← פשוט כותבים בצ'אט.

## איפה כל דבר נמצא

| מה | איפה |
|---|---|
| Custom Instructions | Settings ← Personalization ← Custom Instructions |
| Memory (לראות/למחוק מה הוא יודע) | Settings ← Personalization ← Memory |
| Projects | תפריט צד ← New Project |
| משימות מתוזמנות (Tasks) | פשוט מבקשים בצ'אט: "כל יום ראשון ב-8:00, תעשה X" |
| Deep Research | כפתור Deep Research בתיבת הכתיבה |
| קול | אייקון האוזניות/קול באפליקציה |
| מצלמה | אייקון המצלמה — מכוונים על כל דבר ושואלים |
| GPT Store | תפריט צד ← Explore GPTs. שווה להתקין: TRAVEL (מסלולי טיולים), Construction Estimating Pro, image generator, SAP Expert |
| אפליקציות וחיבורים (Gmail, Spotify, Skyscanner...) | Settings ← Apps and Connectors — או פשוט מזכירים את שם האפליקציה בצ'אט. ⚠️ הרשאות אמיתיות לחשבון האמיתי — לחבר רק מה שבאמת משתמשים בו |

## 6 כללי זהב

1. **לאמת כל דבר שפועלים לפיו** — מספרים, תאריכים, שמות, חישובים. הוא נשמע בטוח גם כשהוא טועה.
2. **אף פעם לא מדביקים מסמכי עבודה סודיים.** מתארים את הבעיה במקום להדביק את המסמך.
3. **נותנים הקשר כמו לבן אדם** — בשביל מי זה, מה תעשו עם זה, מה כבר ניסיתם.
4. **מבקשים שינויים, לא מתחילים מחדש** — "קצר יותר", "בעברית", "בטבלה", "תסביר פשוט יותר".
5. **שואלים על נושא יותר מפעמיים? תעשו ממנו Project.**
6. **ב-GPT Store סומכים כמו באינטרנט** — בודקים את הבונה, תג מאומת, הרבה דירוגים.

## 5 פרומפטים ששווה לגנוב

1. "הנה תמונה של ___ — מה זה ומה כדאי לעשות עם זה?" *(מצלמה)*
2. "מצורף מסמך. הסבר אותו פשוט, ואז תגיד לי אם משהו נראה חריג." *(קבצים)*
3. "תראיין אותי בשאלות, אחת בכל פעם, ואז כתוב ___ לפי התשובות שלי." *(מכתבים, תוכניות, החלטות)*
4. "כל בוקר ב-7:30, שלח לי ___." *(משימות)*
5. "תן לי 3 אפשרויות עם יתרונות וחסרונות, ואז המלץ על אחת והסבר למה." *(כל החלטה)*

## דבר אחד לנסות השבוע

פתחו את ה"תעשו את זה שלכם" בכרטיס המתכון שלכם — או פשוט הוסיפו שורה אחת ל-Custom Instructions שהייתם רוצים שהוא ידע עליכם.
