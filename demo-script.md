# Demo Script — 5 Live Demos

All demos run on ONE demo account (yours or a spare), mirrored to the TV. Exact prompts below — copy-paste, don't improvise live.

---

## Pre-session prep checklist (do 1–2 days before)

- [ ] Demo account is Plus; verify **deep research**, **Projects**, and **scheduled Tasks** all appear in the UI
- [ ] Custom instructions on demo account are **empty** at session start (Demo 1 fills them live)
- [ ] The "after" custom-instructions text (below) saved somewhere you can paste from fast
- [ ] One real PDF ready on the demo device: electricity bill / appliance manual / arnona bill — something the parents recognize
- [ ] Hallucination example (Demo 2b) **tested the day before** — confirm it actually errs; if not, use the fallback
- [ ] Deep research question finalized (Demo 0) — dry-run once to check it takes 10+ min and the output impresses
- [ ] Kayak GPT tested on demo account (Demo 4) — verify it returns real flight results; fallback: Canva GPT ("design a birthday card for grandma")
- [ ] TV screen mirroring tested from the demo device
- [ ] All family devices charged, everyone logged into their OWN ChatGPT account
- [ ] Recipe cards printed (or link ready), cheat sheet printed
- [ ] Full dry-run of Demos 1–3 once, end to end

## Demo 0 (0:00) — Kick off deep research, walk away

**Setup:** big screen, new chat, enable Deep Research mode.

**Prompt (Dad-flavored — adjust destination/dates to something real for your parents):**

> I'm planning a 7-day trip for a couple in their 60s in October. Compare three destinations: Portugal (Lisbon + Porto), northern Greece (Thessaloniki + Halkidiki), and Slovenia (Ljubljana + Lake Bled). For each: weather in October, estimated total cost from Tel Aviv (flights + mid-range hotels + food), how walkable it is for people who prefer light walking, kosher/vegetarian food availability, and one underrated highlight. End with a recommendation and a suggested 7-day itinerary for the winner.

**Say:** "I just asked it something. It'll take a while. Ignore it — we'll come back at the end." Minimize, move on. **Do not** let the room watch it think.

## Demo 1 (0:20) — Custom instructions: before/after

**Part A — before.** New chat, ask:

> Suggest what to make for dinner tonight.

Result: generic list. Say: "Fine. Also useless — it's answering a stranger."

**Part B — set instructions live.** Settings → Personalization → Custom Instructions. Paste (visible on screen, narrate as you paste):

> About me: I live in Israel. I cook for 2 adults in our 60s. We keep kosher, prefer Mediterranean food, and dinner should take under 40 minutes to make. One of us avoids spicy food.
>
> How to answer: Answer in Hebrew unless asked otherwise. Be short and practical. When suggesting recipes, always include a shopping list.

**Part C — after.** NEW chat, same exact question:

> Suggest what to make for dinner tonight.

Result: Hebrew, kosher, 40-minute Mediterranean dinner with a shopping list. **Pause here.** Say: "Same question. I typed nothing extra. It just knows the house now — and it will in every chat from now on."

Mention Memory in one line: "There's also Memory — things it picks up about you over time. Settings → Personalization, you can see and delete everything it remembered."

## Demo 2 (0:30) — File upload + the trust moment

**Part A — upload the PDF** (bill or appliance manual). Ask, in sequence:

> Explain this document to me in simple terms. What am I actually paying for?

> Is there anything here unusual or worth checking?

Pick the follow-up that fits the doc (e.g. manual: "The machine shows error E4 — what do I do, step by step?").

**Say:** "It's not answering from the internet now. It's answering from MY document."

**Part B — the hallucination (trust moment, ~5 min).** New chat, NO web search (disable browsing/tools if possible for this one). Ask:

> What exactly does clause 7.3 of the standard Israeli rental contract law say? Quote it.

Typical result: a confident, official-sounding, **invented** quote (no such standardized clause). Let the room read it. Then say: "Sounds completely real, right? It's invented. It doesn't know, so it produces something that SOUNDS right. That's the single most important thing to know about these tools."

**Fallback** if it refuses/hedges: ask it to list the exact final standings of an obscure old sports season, or exact prices of specific local products — then show the real answer on Google.

**Then the two rules (slide 9 on screen):**
1. Verify anything you'll act on — numbers, dates, names, calculations. Drafting partner, not oracle.
2. Never paste confidential work material. Describe the problem, don't paste the document. (Gently, at Mom: "there's a recipe card built exactly for this.")

## Demo 3 (0:40) — Project + scheduled Task

**Part A — create a Project live.** Sidebar → New Project → name it **"Trip to Portugal"** (callback to Demo 0).

Project instructions (paste, narrate):

> This project is for planning our October trip. We are a couple in our 60s, prefer light walking, kosher food, mid-range budget. Always answer in Hebrew. Keep a running list of open decisions at the end of each answer.

Upload the PDF from Demo 2 or any travel doc if you have one. Then ask inside the project:

> What should we book first, and when?

**Say:** "Every chat inside this folder already knows all of this. Set up once — smart forever. This is what you'll each build in an hour."

**Part B — scheduled Task.** In the same or new chat:

> Every Sunday at 8:00, check flight price trends from Tel Aviv to Lisbon for October and give me a short update in Hebrew: are prices rising or falling, and should we book now.

Show the confirmation that a weekly task now exists. **Say:** "Nobody will be pressing any buttons on Sunday. It runs alone and the answer waits for you. It's not a search engine anymore — it's staff."

## Demo 4 (0:50) — GPT Store: borrow an expert

**Setup:** sidebar → **Explore GPTs** (GPT Store).

**Part A — browse.** Scroll the store on the big screen for 20 seconds — categories, ratings, "By ChatGPT" / verified builders. Say: "159,000+ ready-made expert versions of ChatGPT. Someone probably already built your assistant."

**Part B — use one live.** Search **Kayak** (callback to the trip theme). Open it, ask:

> Find me flight options from Tel Aviv to Lisbon in October for 2 adults, sorted by price.

Show that it answers with real flight data, inside ChatGPT. Then show it's now pinned in the sidebar — "installed, like an app."

**Part C — the menu for the room** (30 seconds, just say it):
- **Kayak** — flights & hotels (Dad)
- **Canva** — design invitations, posters, greeting cards (Mom, sister)
- **Wolfram** — real verified math & engineering calculations (engineer brother)
- **Consensus** — answers backed by scientific papers (health questions — everyone)

**The one rule:** prefer GPTs from companies you know, or with a verified badge + many ratings. Anyone can publish to the store — same trust rules as the rest of the internet.

**Optional one-liner for context:** workplaces are getting a bigger version of this called "Skills" (Business/Enterprise plans) — if you see it at work, it's the same idea.

## Demo 0 reveal (0:58)

Reopen the deep research chat. Scroll SLOWLY through the report — sections, tables, sources. Say: "One hour ago I asked one question. While we talked, it read dozens of sources and wrote this. Imagine what you'd use this for." Perfect segue to break + recipe cards.
