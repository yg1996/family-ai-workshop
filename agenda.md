# Family AI Workshop — Agenda (~2h15)

**Format:** in person, big screen (TV) + everyone on their own phone/laptop.
**Talk language:** Hebrew. Materials: English (recipe cards bilingual).
**Prereq:** everyone logged into their own ChatGPT account before start (Plus where available).

| Time | Segment | Notes |
|---|---|---|
| 0:00–0:05 | Welcome + kick off deep research live | Start the deep research query on the big screen FIRST thing ("we'll come back to this"). See demo-script.md, Demo 0. |
| 0:05–0:15 | The LLM world | What an LLM is (2-min intuition, no math), the providers (OpenAI, Anthropic, Google, Meta), ways to consume (app, voice, camera, built into other products). |
| 0:15–0:20 | "You all use it as a search engine" | The gap: one-off questions vs. an assistant that knows you, reads your stuff, and works for you. Frame the 3 upgrades. |
| 0:20–0:30 | **Upgrade 1: It can know YOU** | Memory + custom instructions. Demo 1: same question, before/after custom instructions. |
| 0:30–0:40 | **Upgrade 2: It can read YOUR stuff** | File upload + deep research. Demo 2: upload a real PDF, ask questions. **Trust moment woven in (~5 min):** live hallucination example, verify numbers/dates, never paste confidential work docs. |
| 0:40–0:50 | **Upgrade 3: It can work FOR you** | Projects + scheduled Tasks. Demo 3: build a Project live (instructions + file), create one scheduled Task. |
| 0:50–0:58 | **Bonus: Borrow other people's experts** | GPT Store + Custom GPTs. Demo 4: browse store, use one pre-built GPT live (Kayak or Canva), pin it to sidebar. "Someone already built your assistant — install it." |
| 0:58–1:03 | Deep research reveal | Return to Demo 0 — "it did all this while we talked." |
| 1:03–1:13 | Break | Hand out recipe cards during break. Coffee. |
| 1:13–1:58 | **Hands-on: pick a recipe, build it** | Each person picks one card, builds on their own device. You float and help. Success = a working Project or Task on THEIR account (or 3 useful GPTs installed — card 8). |
| 1:58–2:13 | Show & tell + wrap | Each person demos their build (1-2 min each). Hand out cheat sheet. "One thing to try this week." |

## Success criteria

- Every person leaves with **one working Project or scheduled Task on their own account** that they'll actually use this week.
- Everyone can say the 3 upgrades in their own words.
- Mom knows the confidentiality rule cold.
