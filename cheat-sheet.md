# ChatGPT — Beyond Questions (Take-Home Cheat Sheet)

## The 3 upgrades (+1 bonus)

1. **It can know YOU** — set custom instructions once; every chat starts personal.
2. **It can read YOUR stuff** — upload PDFs, photos, screenshots; ask questions about THEM.
3. **It can work FOR you** — Projects keep a topic's brain in one place; Tasks run on a schedule without you.
4. **Bonus: Borrow experts** — GPT Store has 159,000+ ready-made assistants. Install with one click.

## The 6 lines everyone should set (Settings → Personalization → Custom Instructions)

Copy, fill the brackets, done — every chat gets better forever:

```text
About me: I live in [city], Israel. I'm not technical. My phone is a [model],
my computer is [Windows/Mac]. [dietary/other constants: e.g. we keep kosher].

How to answer: Always answer in Hebrew. Short and practical. No technical
jargon — if a term is necessary, explain it in one sentence. When giving
instructions, number the steps, ONE action per step, tell me where to tap.
For decisions: 2-3 options with pros and cons, then recommend one. Prices in
shekels. If my question is unclear — ask me before answering.
```

**The sorting rule:** always-true-about-you → instructions. About-today's-question → just type it in the chat.

## 5 golden rules

1. **Verify anything you'll act on** — numbers, dates, names, calculations. It sounds confident even when wrong.
2. **Never paste confidential work documents.** Describe the problem instead of pasting the document.
3. **Give context like you would to a person** — who it's for, what you'll do with it, what you already tried.
4. **Ask for changes, don't start over** — "shorter", "in Hebrew", "as a table", "explain simpler".
5. **If you'll ask about a topic more than twice — make it a Project.**
6. **In the GPT Store, trust like on the internet** — known companies, verified badge, many ratings.

## 5 starter prompts worth stealing

1. "Here's a photo of ___ — what is it and what should I do about it?" *(camera)*
2. "I'm attaching a document. Explain it simply, then tell me if anything looks unusual." *(files)*
3. "Interview me with questions one at a time, then write ___ based on my answers." *(letters, plans, decisions)*
4. "Every morning at 7:30, send me ___." *(tasks)*
5. "Give me 3 options with pros and cons, then recommend one and explain why." *(any decision)*
